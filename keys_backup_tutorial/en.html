1
Use a device as backup device, install BCM on it.
2
On Main Device
“Me” tab → “Account” → “Show Account Key”.
Enter password to show your Account Key QR Code.
On Backup Device
Launch BCM → “More Options” → “Backup Account Key“.
3
Scan the Account Key QR Code shown on main device with the backup device.
4
On Main Device
Tap the button below Account Key QR code to mark as backed up.
Q&A
Q Is this the only way to backup my Account Key？
A Yes. This method is the only and safest way provided. Since it does not involve any server. If you keep the backup device offline, Account Key would be kept from heckers
Q How to login to a new device?
A On the new device, in the launch page, “More options” → “Scan to login”, scan the backed-up Account Key QR Code from the backup device, then enter the password to login
Q Is my wallet account separate from my BCM account?
A No. Your BCM account is your wallet account at the same time.
